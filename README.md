# fetch-pisara
Usage:
```
usage: fetch_pisara.py [-h] [-t {A+,A-,O+,O-,B+,B-,AB+,AB-}] [-f {text,json}]
                       [-l {fi,en}] [--brief]

Get blood supply information in Finland.

optional arguments:
  -h, --help            show this help message and exit
  -t {A+,A-,O+,O-,B+,B-,AB+,AB-}, --type {A+,A-,O+,O-,B+,B-,AB+,AB-}
                        Blood group filter
  -f {text,json}, --format {text,json}
                        Output format
  -l {fi,en}, --lang {fi,en}
                        Output language for text
  --brief               Use simpler output format for text
```
